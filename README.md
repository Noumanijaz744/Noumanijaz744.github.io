# Nouman Ijaz Academic Portfolio

A responsive one-page academic website prepared for GitHub Pages.

## Publish on GitHub Pages

1. Create a repository named `Noumanijaz744.github.io`.
2. Upload every file and folder from this package to the repository root.
3. Open **Settings → Pages** in the repository.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder, then save.
6. The website should appear at `https://noumanijaz744.github.io/` after a few minutes.

## Files

- `index.html`: website content
- `styles.css`: layout, colors, responsive design, dark mode
- `script.js`: mobile navigation, theme switcher, section animation
- `assets/profile.png`: profile photograph from the supplied GitHub profile
- `assets/favicon.svg`: browser icon

## Add a CV later

Place your CV at `documents/Nouman_Ijaz_CV.pdf`, then add this button inside the `hero-actions` block in `index.html`:

```html
<a class="button secondary" href="documents/Nouman_Ijaz_CV.pdf" target="_blank">Download CV</a>
```

## Edit content

Open `index.html` in a text editor. Search for the section name, publication title, email address, or project name that you want to update. No build tools are required.
